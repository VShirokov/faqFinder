# faqFinder
faqFinder for JekaTiger


Реализация задания - "Найти способ предобработки данных с целью достижения алгоритма поиска информации по словам близким к О(1)"

Входные данные имеют вид:

[
    {
        question: 'Вопрос 1',
        answer: 'Ответ 1',
    },
    {
        question: 'Вопрос 2',
        answer: 'Ответ 2',
    }
]
